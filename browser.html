Sobre o Microsoft Edge

Este é um projeto de simulação do navegador Microsoft Edge, desenvolvido para fins educacionais e demonstrativos.

O projeto implementa uma interface similar ao navegador real, com funcionalidades básicas como navegação por abas, histórico de navegação, modo anônimo e pesquisa na web através do Yandex.

Desenvolvido com HTML, CSS e JavaScript puro, sem dependências externas além do Font Awesome para ícones.

Versão: 1.0.0
